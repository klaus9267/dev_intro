오늘의 학습

1. Position 속성의 심층 이해

  • CSS Position 속성은 웹 요소의 배치 방식을 제어하는 핵심 기술
  • 각 Position 값의 고유한 특성과 동작 방식 학습
    • static: 문서의 기본 흐름을 따르는 기본 상태
    • relative: 원래 위치를 기준으로 상대적 이동 가능
    • absolute: 문서 흐름에서 제거되고 가장 가까운 위치 지정 조상을 기준으로 배치
    • fixed: 뷰포트를 기준으로 고정된 위치 유지
    • sticky: 스크롤에 따라 동작이 변하는 하이브리드 포지셔닝

2. z-index의 작동 원리

  • 요소의 쌓임 순서(stacking order)를 제어하는 속성
  • 숫자가 높을수록 상위에 배치
  • position이 static이 아닌 요소에만 적용 가능
  • 음수 값도 사용 가능하여 유연한 레이아웃 구성 가능

3. Flexbox의 혁신적인 레이아웃 모델

  • 1차원 레이아웃 모델로 요소 간 공간 배분과 정렬을 혁신적으로 단순화
  • Flex 컨테이너와 Flex 아이템의 개념 이해
    • Flex 컨테이너: display: flex로 생성
    • Flex 아이템: 컨테이너의 직계 자식 요소

새롭게 알게된 점

1. 레이아웃 설계의 복잡성

  • Position과 Flexbox의 조합으로 더욱 정교한 레이아웃 구현 가능
  • 각 속성의 미묘한 차이가 전체 레이아웃에 큰 영향을 미침

2. 반응형 디자인과의 연관성

  • Flexbox의 유연한 특성이 반응형 웹 디자인에 핵심적인 역할
  • 미디어 쿼리와 결합하여 다양한 디바이스에 최적화된 레이아웃 구현

3. 성능과 최적화 고려사항

  • 과도한 중첩된 Position 사용은 렌더링 성능에 부정적 영향
  • Flexbox의 효율적인 속성 활용으로 성능 최적화 가능
