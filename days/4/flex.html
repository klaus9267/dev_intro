Flexbox란?

Flexbox는 요소들을 행과 열 형태로 배치하는 1차원 레이아웃 모델입니다. 주로 다음과 같은 상황에서 유용합니다:

  • 요소들을 동적으로 정렬하고 분배해야 할 때
  • 요소의 크기가 불명확하거나 동적인 경우
  • 복잡한 레이아웃을 간단하게 구현하고 싶을 때
  • 반응형 디자인을 구현할 때

Flexbox는 부모 요소(Flex Container)와 자식 요소(Flex Items)로 구성되며, 부모 요소에 display: flex를 적용하면 Flex 컨테이너가 생성됩니다.

flex-direction: row

row는 Flexbox의 기본 방향으로, 요소들을 왼쪽에서 오른쪽으로 배치합니다.

주축(main axis)이 가로 방향이 되며, 교차축(cross axis)은 세로 방향이 됩니다.

1
2
3
<div class="flex-container" style="flex-direction: row;"> <div class="flex-item">1</div> <div class="flex-item">2</div> <div class="flex-item">3</div> </div>

flex-direction: column

column은 요소들을 위에서 아래로 배치합니다.

주축이 세로 방향이 되며, 교차축은 가로 방향이 됩니다.

1
2
3
<div class="flex-container" style="flex-direction: column; height: 150px;"> <div class="flex-item">1</div> <div class="flex-item">2</div> <div class="flex-item">3</div> </div>

justify-content: space-around

justify-content는 주축을 따라 Flex Items를 정렬하는 방법을 지정합니다.

space-around는 각 항목의 주변에 동일한 간격을 배치합니다.

  • flex-start: 시작점 정렬
  • center: 중앙 정렬
  • space-between: 양끝 정렬
  • space-around: 균등 간격
1
2
3
<div class="flex-container" style="justify-content: space-around;"> <div class="flex-item">1</div> <div class="flex-item">2</div> <div class="flex-item">3</div> </div>

align-content: space-between

align-content는 여러 줄의 Flex Items를 교차축 방향으로 정렬합니다.

space-between은 첫 줄은 시작점에, 마지막 줄은 끝점에 배치하고 나머지 줄은 균등하게 분배합니다.

이 속성은 flex-wrap: wrap이 설정되어 있고 여러 줄의 아이템이 있을 때만 효과가 있습니다.

A
B
C
D
<div class="flex-container" style="flex-wrap: wrap; align-content: space-between; height: 200px;"> <div class="flex-item">A</div> <div class="flex-item">B</div> <div class="flex-item">C</div> <div class="flex-item">D</div> </div>

flex-wrap: wrap

flex-wrap은 Flex Items가 한 줄에 들어가지 않을 때의 처리 방법을 지정합니다.

  • nowrap(기본값): 한 줄에 모두 표시
  • wrap: 여러 줄로 나눔
  • wrap-reverse: 여러 줄로 나누되 역순으로 배치
1
2
3
4
<div class="flex-container" style="flex-wrap: wrap;"> <div class="flex-item" style="flex-basis: 150px;">1</div> <div class="flex-item" style="flex-basis: 150px;">2</div> <div class="flex-item" style="flex-basis: 150px;">3</div> <div class="flex-item" style="flex-basis: 150px;">4</div> </div>

flex-grow

flex-grow는 Flex Items가 Flex Container의 남는 공간을 어떻게 나눠 가질지 지정합니다.

0보다 큰 값을 지정하면 남는 공간을 해당 비율만큼 차지합니다.

이 예제에서 flex-grow: 2는 다른 항목보다 2배의 여분 공간을 차지합니다.

Grow 2
Grow 1
Grow 0
<div class="flex-container"> <div class="flex-item" style="flex: 2 1 100px;">Grow 2</div> <div class="flex-item" style="flex: 1 1 100px;">Grow 1</div> <div class="flex-item" style="flex: 0 1 100px;">Grow 0</div> </div>

align-items와 justify-content 조합

여러 Flex 속성을 조합하여 더 복잡한 레이아웃을 만들 수 있습니다.

이 예제에서는 justify-content: center로 가로 중앙 정렬을, align-items: flex-end로 세로 하단 정렬을 구현했습니다.

각 아이템의 높이를 다르게 지정하여 시각적인 계층 구조를 만들었습니다.

1
2
3
<div class="flex-container" style="justify-content: center; align-items: flex-end; height: 150px;"> <div class="flex-item" style="height: 50px;">1</div> <div class="flex-item" style="height: 80px;">2</div> <div class="flex-item" style="height: 100px;">3</div> </div>
