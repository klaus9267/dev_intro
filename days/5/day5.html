TIL (Today I Learned)

오늘의 학습

1. AI 도구의 전략적 활용

  • ChatGPT 대안 서비스들의 특징
    • Claude: 프론트엔드와 자연어 처리 강점, artifact 기능
    • Copilot: DALLE 탑재로 이미지 생성 가능
    • Gemini: Imagen 3 모델로 무료 이미지 생성
    • Perplexity: 인터넷 검색과 출처 제시 강점
  • AI 도구는 안경처럼 자연스러운 도구로 활용해야 함

2. Git 핵심 기능과 협업

  • 브랜치 관리와 협업 전략
    • Feature Branch: 팀원이 같은 저장소에서 직접 작업
    • Forking Workflow: 원본 저장소를 개인 계정으로 복제
    • 팀 구성과 프로젝트 성격에 따른 워크플로우 선택
  • Git 유용한 기능들
    • stash: 작업 중인 내용 임시 저장
    • cherry-pick: 특정 커밋만 선택적 적용
    • amend: 최근 커밋 수정

3. 개발자의 성장 방향

  • 효율적인 프로젝트 관리
    • 하루 1시간 원칙: 지속 가능한 개발 습관
    • 실현 가능한 목표 설정의 중요성
  • 기존 도구와 템플릿의 효과적 활용
  • 욕심내지 말고 단계적 성장 추구

새롭게 알게된 점

1. 프로젝트 관리의 실제

  • 시간 기반 프로젝트 관리
    • 하루 1시간은 책임감으로 가능
    • 2시간부터는 지속하기 어려움
  • MVP 중심의 개발 접근 방식

2. 헤드리스 CMS의 활용

  • Strapi, Superbase, Firebase 등 현대적 도구
  • 백엔드 구현 없이 프로토타입 개발 가능

3. 포트폴리오 전략

  • 템플릿 활용이 부끄러운 것이 아님
  • AI 도구를 활용한 효율적 개발
  • 결과물의 품질과 완성도 중시
