TIL (Today I Learned)

오늘의 학습

1. HTML 기본 구조와 시맨틱 태그의 중요성

  • HTML은 단순한 마크업 언어를 넘어서 웹의 구조와 의미를 전달하는 중요한 역할을 합니다.
  • 시맨틱 태그(header, nav, section, footer 등)는 검색 엔진 최적화(SEO)와 접근성 향상에 큰 도움이 됩니다.

2. Favicon과 OG 태그의 활용

  • Favicon은 브라우저 탭에 표시되는 아이콘으로, 사이트의 브랜딩에 중요합니다.
  • OG 태그는 SNS 공유 시 콘텐츠 미리보기를 제어하는 메타태그로, 마케팅적 요소가 있습니다.

3. CSS 기초와 선택자 시스템 이해

  • CSS는 캐스케이딩 원칙을 따르며, 선택자의 우선순위가 매우 중요합니다.
  • 선택자의 명시도 계산법: ID(100점), 클래스(10점), 요소(1점)

새롭게 알게된 점

1. 백엔드 개발자 관점에서의 HTML 중요성

  • RESTful API 응답으로 보내는 데이터가 프론트엔드에서 어떻게 렌더링되는지 이해하는데 도움
  • SEO 최적화를 위해서는 백엔드에서도 적절한 메타 태그 관리가 필요

2. 성능 최적화 관련

  • CSS 선택자의 복잡도가 렌더링 성능에 영향을 미칠 수 있음
  • 백엔드에서 제공하는 데이터 구조가 프론트엔드의 CSS 선택자 설계에 영향

3. 보안 관련

  • CSS 인젝션 공격 가능성에 대한 이해
  • CSP(Content Security Policy) 설정의 중요성
