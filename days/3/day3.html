TIL (Today I Learned)

오늘의 학습

1. CSS의 기본 개념과 작동 원리

  • CSS는 웹 페이지의 시각적 표현을 담당하는 스타일 시트 언어
  • HTML 문서의 레이아웃과 디자인을 제어하는 핵심 기술
  • 콘텐츠와 디자인의 분리를 통해 유지보수성 향상

2. CSS 선택자와 명시도 시스템

  • 선택자의 유형: 요소, 클래스, ID, 속성 선택자
  • 명시도 계산 방법 심층 분석
    • ID 선택자: 100점
    • 클래스 선택자: 10점
    • 요소 선택자: 1점
  • 캐스케이딩 원칙의 중요성 이해

3. 박스 모델과 레이아웃 이해

  • CSS 박스 모델의 구성 요소 분석
    • Content: 실제 내용 영역
    • Padding: 내용과 테두리 사이 여백
    • Border: 요소의 테두리
    • Margin: 요소 간 간격
  • `box-sizing` 속성의 동작 원리 (`content-box` vs `border-box`)

새롭게 알게된 점

1. CSS의 성능 최적화 관점

  • 선택자의 복잡성이 렌더링 성능에 미치는 영향
  • 효율적인 CSS 선택자 작성의 중요성
  • 불필요한 중첩 선택자 피하기

2. 반응형 웹 디자인 기초

  • `display` 속성의 다양한 활용
    • `block`, `inline`, `inline-block` 각각의 특징
  • 미디어 쿼리를 통한 반응형 레이아웃 구현 기본

3. 크로스 브라우저 호환성

  • CSS 리셋과 노말라이즈의 중요성
  • 벤더 프리픽스를 통한 브라우저 호환성 대응

앞으로 학습할 내용

  1. 고급 CSS 레이아웃 기법 (Flexbox, Grid)
  2. CSS 전처리기 (SASS, LESS) 심층 학습
  3. CSS 성능 최적화 전략
  4. 모던 CSS 트렌드와 새로운 기능들
